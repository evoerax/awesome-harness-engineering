# OpenClaw

> Community-driven AI coding agent harness.

## Overview

OpenClaw is a skill-first harness with community plugin support. Uses `clawhub` for skill installation.

## Key Features

| Feature | Details |
|---------|---------|
| **Hooks** | ❌ |
| **Skills** | SKILL.md format, installed via `clawhub` |
| **Config** | `~/.openclaw/` |
| **Community** | Community-driven plugin ecosystem |

## Installation

```bash
clawhub install pua
# or
mkdir -p ~/.openclaw/skills/pua
curl -o ~/.openclaw/skills/pua/SKILL.md ...
```
